URL Shortener Microservice

API Basejump: URL shortener microservice

User stories:
    1) I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
    2) If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
    3) When I visit that shortened URL, it will redirect me to my original link.

Creation

Example usage:

https://fcc-backend-urlshortener-slkinnison.c9users.io/api/new/https://www.google.com
https://fcc-backend-urlshortener-slkinnison.c9users.io/api/new/http://freecodecamp.com/news
If you want to pass a site that doesn't exist (or an invalid url) for some reason you can do:
https://fcc-backend-urlshortener-slkinnison.c9users.io/api/new/invalid?allow=true

Example output:

{ "original_url": "http://freecodecamp.com/news", "short_url": "https://fcc-backend-urlshortener-slkinnison.c9users.io/api/7" }

Redirection

Example usage:

https://fcc-backend-urlshortener-slkinnison.c9users.io/api/7
Will redirect to
http://freecodecamp.com/news
